### Overview
A User represents a person using the system.

- [[Last Name]]

### Capabilities
A User may:
- create Commitments
- post to Commitments
- subscribe to Commitments

### Relationships
- A User may own zero or more Commitments
- A User may create zero or more Posts
- A User may subscribe to zero or more Commitments
